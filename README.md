A modern, clean, and intuitive task management application designed for productivity. Built with a focus on simplicity, this board allows users to organize their workflow with seamless drag-and-drop interactions.

## 🚀 Features
- **Modern SaaS Design:** A clean, professional aesthetic using deep slates and indigo accents.
- **Kanban Workflow:** Intuitive drag-and-drop interface for managing tasks.
- **Persistent Storage:** Your tasks are automatically saved to your browser's local storage.
- **Responsive UI:** Clean typography and spacious layout for better focus.

## 🛠 Tech Stack
- **HTML5:** Semantic structure.
- **CSS3:** Modern styling (Flexbox/Grid, transitions, and depth effects).
- **JavaScript (Vanilla):** Logic for drag-and-drop and state management.

## 📦 Installation
1. Clone this repository or download the source files.
2. Open `index.html` in any modern web browser.
3. Start adding tasks!

## 🎨 Design Philosophy
This application utilizes a "SaaS-minimalist" approach:
- **Rounded Corners:** 8px radius for a softer, modern look.
- **Depth:** Subtle shadows give task cards a floating effect.
- **Typography:** Uses system-ui fonts for high legibility.
- **Color Palette:** High-contrast indigo accents against deep slate backgrounds.

## 💡 Usage
- **Add:** Type a task in the input field and hit enter to add to "To Do."
- **Move:** Click and drag any task card to a different column.
- **Persistence:** Refresh the page anytime—your tasks will remain exactly where you left them.
